# Another Red Online

포켓몬 팬게임 **Another Red**에 **온라인 1:1 대전(PvP)** 을 추가하는 모드입니다.
게임을 재설치하거나 다시 컴파일할 필요 없이, 배포 zip을 게임 폴더에 덮어쓰기만
하면 적용됩니다. 친구와 방 코드를 주고받아 실시간으로 대전할 수 있습니다.

> 이 모드는 대전 중에만 포켓몬을 규격에 맞게 복제해서 사용하며, **여러분의
> 세이브 파일(레벨·개체값 등)은 전혀 바뀌지 않습니다.**

---

## 요구 사항

- **Another Red (PWT / 250821 빌드)** — 이 모드는 해당 베이스 게임 빌드에 맞춰
  제작되었습니다. 다른 빌드에서는 정상 동작을 보장하지 않습니다.
- 대전 상대와 **같은 버전의 모드**를 설치해야 합니다(아래 *버전 주의* 참고).
- 인터넷 연결.

## 설치

1. [Releases](../../releases)에서 최신 `AnotherRedOnline_vX.Y.Z.zip`을 내려받습니다.
2. 압축을 풀면 `Data`, `Audio` 폴더가 들어 있습니다.
3. 이 폴더들을 **`Game.exe`가 있는 Another Red 게임 폴더**에 그대로 붙여넣고,
   덮어쓰기(파일 병합/교체)를 선택합니다.
4. 끝입니다. 게임을 실행하세요.

업데이트도 동일합니다 — 새 zip을 같은 방법으로 덮어쓰면 됩니다(이전 버전을
지울 필요 없음).

## 대전하는 방법

1. 게임 안에서 아무 **PC**(포켓몬 센터 등)에 접근합니다.
2. PC 메뉴에서 **「온라인 대전」** 을 선택합니다.
3. 방을 만들거나 입장합니다.
   - **방 만들기(호스트):** 코드 입력란을 비운 채 확인 → 포맷을 고르면 **방 코드**가
     표시됩니다. 이 코드를 상대에게 알려주세요.
   - **입장(게스트):** 상대가 알려준 **방 코드**를 입력합니다(영문은 자동으로
     대문자 처리됩니다).
4. 두 사람이 연결되면 팀 선출/대전이 자동으로 시작됩니다. 취소 키로 언제든
   중단할 수 있습니다.

## 대전 포맷

| 포맷 | 방식 | 선출 |
|------|------|------|
| **싱글 배틀** | 1:1 싱글 배틀 | 6마리 중 **3마리** 선출 |
| **더블 배틀** | 2:2 더블 배틀 | 6마리 중 **4마리** 선출 |

포맷은 방을 만드는 사람(호스트)이 정하며, 게스트는 방의 포맷을 따릅니다.
두 포맷 모두 대전 전 **오픈 시트 선출**(서로의 팀을 보고 고르는) 화면이 나옵니다.

## 대전 규칙 (중요)

공정한 대전을 위해, 대전에 나가는 **모든 포켓몬은 자동으로 아래 규격에 맞춰집니다.**

- **레벨: 전원 Lv. 50 으로 통일.**
- **개체값(IV): 6개 능력치 전부 최대치(31), 즉 6V 로 통일.**
  전송된 개체값은 무시하고 강제로 최대치로 맞춥니다(치트 방지). → **레벨을 50으로
  맞추면서 개체값도 자동으로 전부 최대가 됩니다. 따로 준비할 필요가 없습니다.**
- **노력치(EV): 여러분 포켓몬의 값을 그대로 사용합니다**(자동으로 최대가 되지는
  않습니다). 성격 · 특성 · 기술 · 지닌 도구도 그대로 유지됩니다.
- 그 밖의 규칙: **같은 종 중복 금지**, **잠듦 유발 기술은 한 팀당 하나만(슬립 클로즈)**,
  **대전 중 가방 아이템 사용 불가.**

### 제한 시간 (체스 클록)

- 플레이어당 **총 7분**의 시간 은행을 가집니다. 이 시간은 **자신이 행동을 고르는
  동안에만** 줄어듭니다(연출/애니메이션 중에는 멈춤).
- **한 턴 선택은 최대 45초**입니다. 시간을 넘기면 자동으로 기술이 선택됩니다.
  (더블 배틀에서는 두 포켓몬의 행동 결정이 이 45초를 **함께 공유**합니다.)
- **선출 화면은 90초** 제한입니다.
- 시간 은행을 모두 소진하면 해당 턴 종료 시 패배합니다.

## 버전 주의

- 대전하는 **두 사람의 모드 버전이 서로 같아야 합니다.** 버전이 다르면 접속 시
  「모드 버전이 상대와 다릅니다」라는 안내와 함께 대전이 거부됩니다(대전 도중
  동기화가 깨지는 것을 막기 위한 안전장치입니다).
- 이 모드는 자동 업데이트가 없으므로, 새 버전이 나오면 **두 사람 모두** 최신 zip을
  덮어써 주세요.

## 문제가 생기면

- 대전 중 오류가 발생하면 게임 폴더(또는 mkxp 저장 경로)에 `arnet_err.txt`
  로그가 남습니다. 문제 제보 시 이 파일을 함께 첨부해 주세요.
- 방 코드를 입력했는데 「잘못된 코드입니다」가 나오면 코드가 정확한지, 상대가 아직
  방을 열어 두고 있는지 확인하세요.
