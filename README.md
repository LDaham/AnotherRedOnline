# Another Red Online

포켓몬 팬게임 **Another Red**에 **온라인 1:1 대전(PvP)** 을 추가하는 모드입니다.
게임을 재설치하거나 다시 컴파일할 필요 없이, 배포 zip을 게임 폴더에 덮어쓰기만
하면 적용됩니다. 친구와 방 코드를 주고받아 실시간으로 대전할 수 있습니다.

> 이 모드는 대전 중에만 포켓몬을 규격에 맞게 복제해서 사용하며, **여러분의
> 세이브 파일(레벨·개체값 등)은 전혀 바뀌지 않습니다.**

---

## 주요 기능 및 변경 사항

**온라인 멀티 대전**
- 1:1 온라인 대전 (호스트 + 게스트 구조)
- 방 코드 대전 / 랜덤 매칭
- **배틀 기믹 지원** — 메가진화 · 다이맥스 · 테라스탈 · Z기술 · 울트라버스트를 온라인
  대전에서 사용할 수 있으며, 양쪽 화면에 동일하게 동기화되어 표시됩니다.

**포켓몬 챔피언스 밸런스 패치 적용**
- 메가진화 A그룹·B그룹(M-A, M-B) 적용 완료
- 앞으로도 밸런스 패치를 지속적으로 반영할 예정
- ※ 어나더레드 ZA 메가진화 패치는 불안정 요소가 많아 현재 추가하지 않았습니다.

**기존 게임 오류 수정**
- 쾌청, 잠듦 즉시 기상, 원시 날씨, 이젝트 팩, 특성 1.3배, 아이템 증발 등의 버그 수정
- 메가진화 포켓몬의 종족값 오류 수정
- NPC 노력치 오류 수정 (EV 3253개 + IV 3280개 항목에 적용)
- 아머까오 택시 루프 오류 수정
- 여러가지 텍스트 표시 오류 수정

**기술 효과(상성) 표시**
- 기술의 효과(대미지 배율)를 특수 기호로 표시합니다.

  | 배율 | 표시 | 의미 |
  |------|:----:|------|
  | 4배   | ☆ | 효과가 매우 굉장함 |
  | 2배   | ◎ | 효과가 굉장함 |
  | 1배   | ○ | 효과 있음 |
  | 0.5배 | △ | 효과가 별로 |
  | 0.25배| ▽ | 효과가 매우 별로 |
  | 0배   | ✕ | 효과 없음 |

**능력치 변화 표시**
- 능력치가 변한 포켓몬에 `[능력치] △△`처럼 누적 변화 정도를 표시합니다.

**1회용 도구 유지**
- 기력의 띠 같은 1회성 도구가 대전이 끝나면 복구됩니다.

**진화의 휘석 장착 시 진화 방지**
- 진화의 휘석을 지니고 있는 포켓몬은 진화하지 않습니다.

**노력치(EV) 표시**
- 스탯 창에서 노력치를 확인할 수 있습니다. 포켓몬 챔피언스 방식의 **능력 포인트(SP)**
  로 표시됩니다(1 SP = 8 EV, 스탯당 최대 32, 합계 최대 66).

**노력치(EV) 조정 — 포켓몬 센터 부가 기능**
- **포켓몬 센터의 부가 서비스 NPC**(미라클 교환 아래 「노력치 조정」)에서 포켓몬의
  노력치를 자유롭게 재분배할 수 있습니다.
- 포켓몬 챔피언스 방식으로 배분합니다 — **포켓몬당 66 SP, 스탯당 최대 32 SP**
  (1 SP = 8 EV). 같은 화면에서 **성격도 변경**할 수 있습니다.
- 조정 화면의 능력치는 **대전 기준(Lv.50 · 6V)** 으로 미리 표시되며, 선택한 성격에
  따라 오르는 스탯은 **붉게**, 내려가는 스탯은 **파랗게** 표시되어 배분 결과를 바로
  확인할 수 있습니다.

**백화점에서 희귀 볼 판매**
- 기존에 구할 수 없던 울트라볼(비스트볼), 드림볼, 사파리볼 등을 백화점에서 구매할 수 있습니다.

## 요구 사항

- **[Another Red (PWT / 250821 빌드)](https://pokemon.jhcloud.kr/)** — 이 모드는
  해당 베이스 게임 빌드에 맞춰 제작되었습니다. 다른 빌드에서는 정상 동작을 보장하지
  않습니다.
- 대전 상대와 **같은 버전의 모드**를 설치해야 합니다(아래 *버전 주의* 참고).
- 인터넷 연결.

## 설치

1. [Releases](../../releases)에서 최신 `AnotherRedOnline_vX.Y.Z.zip`을 내려받습니다.
2. 압축을 풀면 `Data`, `Audio` 폴더가 들어 있습니다.
3. 이 폴더들을 **`Game.exe`가 있는 Another Red 게임 폴더**에 그대로 붙여넣고,
   덮어쓰기(파일 병합/교체)를 선택합니다.
4. 끝입니다. 게임을 실행하세요.

업데이트도 동일합니다 — 새 zip을 같은 방법으로 덮어쓰면 됩니다(이전 버전을
지울 필요 없음).

## 대전하는 방법

1. 게임 안에서 아무 **PC**(포켓몬 센터 등)에 접근합니다.
2. PC 메뉴에서 **「온라인 대전」** 을 선택합니다.
3. **대전 방식**을 고릅니다 — **「방 만들기 / 방 참가 / 랜덤 매칭」**.
   - **방 만들기(호스트):** 대전 포맷을 고르면 **방 코드**가 표시됩니다. 이 코드를
     상대에게 알려주세요. 상대가 입장하면 자동으로 시작됩니다.
   - **방 참가(게스트):** 상대가 알려준 **방 코드**를 입력합니다(영문은 자동으로
     대문자 처리됩니다).
   - **랜덤 매칭:** 대전 포맷을 고르면 같은 포맷을 고른 다른 플레이어와 자동으로
     매칭됩니다(방 코드가 필요 없습니다).
4. 두 사람이 연결되면 팀 선출/대전이 자동으로 시작됩니다. 취소 키로 언제든
   중단할 수 있습니다.

## 대전 포맷

| 포맷 | 방식 | 선출 |
|------|------|------|
| **싱글 배틀** | 1:1 싱글 배틀 | 6마리 중 **3마리** 선출 |
| **더블 배틀** | 2:2 더블 배틀 | 6마리 중 **4마리** 선출 |

포맷은 방을 만드는 사람(호스트)이 정하며, 게스트는 방의 포맷을 따릅니다.
두 포맷 모두 대전 전 **오픈 시트 선출**(서로의 팀을 보고 고르는) 화면이 나옵니다.

## 대전 규칙 (중요)

공정한 대전을 위해, 대전에 나가는 **모든 포켓몬은 자동으로 아래 규격에 맞춰집니다.**

- **레벨: 전원 Lv. 50 으로 통일.**
- **개체값(IV): 6개 능력치 전부 최대치(31), 즉 6V 로 통일.**
  전송된 개체값은 무시하고 강제로 최대치로 맞춥니다(치트 방지). → **레벨을 50으로
  맞추면서 개체값도 자동으로 전부 최대가 됩니다. 따로 준비할 필요가 없습니다.**
- **노력치(EV): 여러분 포켓몬의 값을 그대로 사용합니다**(자동으로 최대가 되지는
  않습니다). 성격 · 특성 · 기술 · 지닌 도구도 그대로 유지됩니다.
- 그 밖의 규칙: **같은 종 중복 금지**, **같은 도구 중복 금지**, **대전 중 가방 아이템 사용 불가.**

### 제한 시간 (체스 클록)

- 플레이어당 **총 7분**의 시간 은행을 가집니다. 이 시간은 **자신이 행동을 고르는
  동안에만** 줄어듭니다(연출/애니메이션 중에는 멈춤).
- **한 턴 선택은 최대 45초**입니다. 시간을 넘기면 자동으로 기술이 선택됩니다.
  (더블 배틀에서는 두 포켓몬의 행동 결정이 이 45초를 **함께 공유**합니다.)
- **선출 화면은 90초** 제한입니다.
- 한쪽이 시간 은행을 모두 소진하면 해당 턴 종료 시 패배합니다.
- **양쪽이 같은 턴에 모두 시간을 소진하면 무승부**가 됩니다. 포켓몬 챔피언스 규칙에
  따라, 시간 종료 시에는 남은 포켓몬 수나 HP를 따지지 않고 무승부로 처리합니다.

## 버전 주의

- 대전하는 **두 사람의 모드 버전이 서로 같아야 합니다.** 버전이 다르면 접속 시
  「모드 버전이 상대와 다릅니다」라는 안내와 함께 대전이 거부됩니다.
- 이는 단순한 편의가 아니라 **공정한 대전을 위한 무결성 보호**입니다. 버전·파일이
  일치해야 하는 이유:
  - **대전 도중 동기화(결정론 락스텝)가 깨지는 것을 방지** — 두 사람의 게임이 완전히
    같은 규칙·데이터로 계산해야 대전이 어긋나지 않습니다.
  - **존재 불가능한 포켓몬(존불몬) 사용 방지** — 데이터를 변조해 원래 배울 수 없는
    기술을 가진 포켓몬 등은 상대의 게임이 검증하여 대전을 거부합니다.
  - **변조·패치된 모드/게임 파일 차단** — 모드 파일이 상대와 일치하지 않으면 대전이
    시작되기 전에 차단됩니다.
- 이 모드는 자동 업데이트가 없으므로, 새 버전이 나오면 **두 사람 모두** 최신 zip을
  덮어써 주세요.

## 문제가 생기면

- 대전 관련 로그는 게임 폴더(또는 mkxp 저장 경로)의 **`ARNet_Logs` 폴더** 안에
  모입니다. 오류 로그(`arnet_err.txt`)를 비롯해 대전 이벤트·제한 시간 기록 등이
  이 폴더에 저장되므로, 문제 제보 시 **`ARNet_Logs` 폴더째로** 첨부해 주시면 됩니다.
- 방 코드를 입력했는데 「잘못된 코드입니다」가 나오면 코드가 정확한지, 상대가 아직
  방을 열어 두고 있는지 확인하세요.
